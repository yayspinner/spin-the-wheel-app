# Random Wheel Picker (React + Vite)

A lightweight random wheel picker built with React and Vite. This project provides a simple interface to create a spinning wheel with custom entries and generate a random result in a visual and interactive way.

## Overview

This repository is a minimal implementation of a “spin the wheel” concept using modern frontend tools. It focuses on clarity and simplicity, making it easy to understand how the logic works and how the UI is structured.

The goal is to offer a clean starting point for developers who want to build their own random picker, game mechanic, or interactive component.

## Live Example

If you prefer a ready-to-use version with more features and a polished UI, you can try it here:  
👉 https://yayspinner.com/

## Key Features

- Add and manage custom entries  
- Smooth spinning animation with randomized result  
- Built with React + Vite for fast performance  
- Simple structure, easy to extend  
- Runs entirely in the browser
